#  To-Do Task Manager (CLI)

A simple command-line task manager built in Python to add, view, and remove daily tasks.  
This is my first backend-oriented CLI project from my 8-month internship roadmap.

---

##  Features
- Add new tasks  
- View all tasks  
- Remove specific tasks  
- Input validation and clean UI  
- Graceful exit message

---

##  Run Locally
```bash
git clone https://github.com/shreyanshrastogi/To-Do-Task-Manager.git
cd To-Do-Task-Manager
python to_do_manager.py
